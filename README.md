## 🌟 About This Project

This state-of-the-art portfolio is engineered from the ground up using modern web technologies to deliver a flawless user experience. Built to highlight specialized technical skills, academic history, certifications, and high-impact web/cybersecurity projects. 

### ✨ Key Features
- **Modern UI/UX**: Distinct Glassmorphism aesthetic with floating micro-interactions.
- **Native Dark/Light Mode**: Synced seamlessly to the user's OS preference (`prefers-color-scheme`) with a manual fallback option.
- **Animation Heavy**: Powered by Framer Motion for smooth scroll effects and dynamic element rendering.
- **Form Integration**: Fully functioning "Contact Me" form directly routed via the EmailJS API.
- **Downloadable Assets**: Direct dynamic linking for PDF CVs and Resume distribution.
- **Mobile-First**: Flawless responsive scaling across mobile, tablet, and desktop viewports.

---

## 🛠️ Technology Stack

- **Frontend Framework:** React.js (v19)
- **Styling:** Tailwind CSS (v4)
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Form Delivery:** EmailJS API (REST Fetch Method)
- **Build Tool:** Vite

---

## 📂 Project Architecture

```text
src/
├── assets/          # Static documents (PDF Resume, profile imagery)
├── components/      
│   ├── ui/          # Reusable micro-components (Buttons, Cards, ThemeToggle)
│   └── sections/    # Macro page sections (Hero, About, Experience, Projects)
├── App.jsx          # Primary router and layout orchestrator
├── main.jsx         # DOM entry point
└── index.css        # Root style definitions and Tailwind directives
```

## 🚀 Getting Started

To run this repository locally on your machine:

### 1. Clone the repository
```bash
git clone https://github.com/jashwanthreddy21/Portfolio.git
cd Portfolio
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

### 4. Build for Production
```bash
npm run build
```
